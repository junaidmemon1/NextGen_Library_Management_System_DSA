# 📚 NextGen – Premium Library Management System

A modern and interactive **Library Management System** built with **React.js**, **Tailwind CSS**, and **Firebase Authentication**.

This project demonstrates how powerful frontend development can become when combined with core **Data Structures and Algorithms (DSA)** concepts.

Unlike traditional library systems that heavily rely on databases for every operation, **NextGen** manages all book-related functionalities using a custom-built **Singly Linked List** implementation in JavaScript.

From adding books to issuing and returning them — every operation is handled dynamically through optimized DSA logic.

The project delivers a clean UI, secure authentication system, and real-time book management experience, making it both educational and practical for modern web development learning.

# 🧠 Data Structures & Algorithms Implementation

One of the core goals of this project is to implement real-world operations using custom DSA logic instead of relying on databases.

## 📌 Singly Linked List

The entire book management system is powered by a custom **Singly Linked List**.

### Implemented Operations

| Method | Description |
|--------|-------------|
| `push()` | Adds a new book to the library |
| `pop()` | Removes the last book from the library |
| `traverse()` | Displays all available books |
| `findByTitle()` | Searches books using title |
| `findByAuthor()` | Searches books using author name |
| `issueBook()` | Marks a book as issued |
| `returnBook()` | Marks a book as returned |

---

# 🔍 Searching Mechanism

The project uses **Linear Search** within the linked list structure to efficiently locate books by:

- 📚 Book Title
- ✍️ Author Name

Search results are displayed dynamically in real-time.
